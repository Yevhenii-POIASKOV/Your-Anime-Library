# Your Anime Library (YAL)

Веб-орієнтована система для створення та ведення персональної бібліотеки аніме-тайтлів. Проєкт розроблено в рамках навчальної дисципліни «Моделювання та аналіз програмного забезпечення».

## Про проєкт

**Your Anime Library** призначений для розв'язання проблеми втрати історії переглядів на різних стрімінгових платформах. Вебсайт дозволяє збирати, структурувати та зберігати інформацію про всі ваші улюблені тайтли в одному зручному місці, додавати персональні оцінки та нотатки.

### Основна мета

Зберігати в єдиному сховищі історію переглядів, поточні онгоїнги та заплановані до перегляду тайтли із фіксацією їхнього поточного статусу.

---

## Технологічний стек

* **Платформа:** Веб-застосунок
* **Frontend:** React (JavaScript) — побудова динамічного та інтерактивного інтерфейсу користувача.
* **Backend:** Node.js — обробка API-запитів.
* **База даних:** MySQL — збереження даних користувачів, списків аніме, статусів та оцінок.
